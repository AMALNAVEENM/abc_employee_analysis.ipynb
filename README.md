# ABC Company Employee Analysis

Comprehensive analysis of 458 employees across teams, positions, age, and salary.

## Tasks Completed
1. Fixed `height` column with random values (150–180 cm)
2. Team distribution (%) 
3. Position segregation
4. Predominant age group
5. Highest salary expenditure (team & position)
6. Age-salary correlation

## Visualizations
- Bar charts for team/position
- Age group histogram
- Salary scatter plot with regression line
